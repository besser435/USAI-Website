Bug Bounty

USAI offers bounties for bugs found. Bounties will be paid with in game currency. Bugs can be on the USAI website/API, Discord bot, or even on the TEAW server itself. Even simple bugs like typos will be rewarded!

Please report security related bugs to brandonusa on Discord in a direct message.

Feature Requests

If you have a simple feature that you want added to any USAI service, drop it in the USAI Discord server. If it's a good idea, we'll add it and reward you with in game currency.

Contributing Code

This website is open source. You can view the code on GitHub here. If you want to contribute code, you can submit a pull request on GitHub. We are mainly looking for visual improvements, as the website kinda looks terrible right now. If your pull request is accepted, you will be rewarded with in-game currency. Please note that some USAI services and APIs are not open source.
